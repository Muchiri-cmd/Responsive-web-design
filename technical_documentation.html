Introduction

Python is a widely used general-purpose, high level programming language. It was created by Guido van Rossum in 1991 and further developed by the Python Software Foundation. It was designed with an emphasis on code readability, and its syntax allows programmers to express their concepts in fewer lines of code.Python is versatile, user-friendly and known for its simplicity, readability, and extensive libraries. It supports multiple programming paradigms and finds applications in web development, data analysis, machine learning, and more. With a vibrant community and cross-platform compatibility, Python is an ideal choice for both beginners and experienced developers.

Python 2.0 and Python 3.0

Python 2.0 and Python 3.0 are two major versions of the Python programming language with notable differences.Python 2.0 reached its end-of-life, and developers are encouraged to transition to Python 3.x versions.

The key differences between the two are:

  • Print: Python 2.0 uses a print statement, while Python 3.0 uses a print() function
  • Division: Python 2.0 performs integer division by default, while Python 3.0 always performs true division
  • String Handling: Python 3.0 eliminates the distinction between ASCII and Unicode strings.
  • Print Compatibility: Some print statements from Python 2.0 may need modifications to work in Python 3.0.
  • Iteration: Python 3.0 introduces a more memory-efficient iteration syntax.
  • Function Annotations: Python 3.0 allows attaching metadata to function arguments and return values
  • Exception Handling: Python 3.0 modifies the syntax for catching exceptions.

Variables

In Python, variables are named identifiers used to represent values in your code. Identifiers start with a letter or underscore, followed by letters, digits, or underscores. Python is case-sensitive. Unicode characters and escape sequences can be used. Reserved keywords such as if,for,while,def,class are excluded.

Declaring Variables

To declare variables in Python, you don't need to explicitly specify their data types. Python uses dynamic typing, which means that variables are assigned values based on the type of data they hold.Python variables can hold different types of values, such as numbers, strings, lists, dictionaries, etc. You don't need to explicitly declare the type; it is inferred based on the assigned value

Variable Assignment: To declare and assign a value to a variable, use the assignment operator (=).Its similar for constants and variable reassignment For example:

              
                my_variable=10
                my_variable=20 #Re-assignment
                PI=3.142 #constant

NB:The # indicates comments in python.This part is ignored by compiler in code execution.It only acts as a reference for human.

Multiple Assignments: Python allows you to assign multiple variables in a single line. For example:

x=y=z=0

Assigning Different Values to Multiple Variables: You can assign different values to multiple variables in a single line. For example:

a,b,c=1,2,3

Data Types

In Python, there are several data types that are commonly used:

  1. Boolean: Represents the truth values True and False
  2. None: Similar to the null value in other languages, it represents the absence of a value.
  3. Numeric Types:
    • Integer: Represents whole numbers, e.g, 42.
    • Float: Represents decimal numbers, e.g, 3.14159.
    • Complex: Represents numbers with a real and imaginary part,e.g, 2 + 3j.
  4. String: Represents a sequence of characters enclosed in single quotes ('') or double quotes ("")
  5. List: Represents an ordered collection of items. Lists can contain elements of different data types and are mutable (modifiable).They are expressed with []
  6. Tuple: Similar to lists, tuples are ordered collections of items. However, tuples are immutable (unchangeable).They are expressed by ( )
  7. Set: Represents an unordered collection of unique elements. Sets do not allow duplicate value.Expressed by set()
  8. Dictionary: Represents key-value pairs. Each element in a dictionary consists of a key and its associated value.Expressed by {}

Operators

In Python, you have various operators that allow you to perform different operations on data. Here are some key operators in Python:

  1. Arithmetic Operators
    • Addition (+)
    • Subtraction (-)
    • Multiplication (*)
    • Division (/)
    • Floor Division (//)
    • Modulo (%)
    • Exponentiation (**)
  2. Comparison Operators
    • Equal to (==)
    • Not equal to (!=)
    • Greater than (>)
    • Less than (<)
    • Greater than or equal to (>=)
    • Less than or equal to (<=)
  3. Assignment Operators
    • Assign (=)
    • Add and assign (+=)
    • Subtract and assign (-=)
    • Multiply and assign (*=)
    • Divide and assign (/=)
    • Modulo and assign (%=)
    • Exponentiation and assign (**=)
  4. Logical Operators
    • Logical AND (and)
    • Logical OR (or)
    • Logical NOT (not)
  5. Bitwise Operators
    • Bitwise AND (&)
    • Bitwise OR (|)
    • Bitwise XOR (^)

    • Right shift (>>)
  6. Membership Operators
    • in: Returns True if a value is found in a sequence.

    • is: Returns True if two variables refer to the same object.
    • is not: Returns True if two variables do not refer to the same object

Lists and Tuples

Lists and tuples are two fundamental data structures in Python. A list is a mutable collection of items enclosed in square brackets, while a tuple is an immutable collection enclosed in parentheses.

Lists:

Lists are commonly used when we need to store and modify a collection of elements. For example, consider a list of numbers:

numbers=[1,2,3,4,5]
We can access individual elements using their index, such as numbers[0] to retrieve the first element. Lists can hold different data types and can be modified by adding or removing elements using methods like append() or remove().

Tuples:

On the other hand, tuples are used when we want to store a fixed collection of items that should not be changed.Tuples are useful in situations where data integrity is important, as they cannot be modified once created. Although tuples have fewer methods compared to lists, they can be indexed and sliced similar to lists. For instance, a tuple of coordinates:

co-ordinates=(10,20)

In summary, lists are mutable, versatile, and suited for dynamic data, while tuples are immutable, used for static data, and offer data protection.

if..else statement

In Python, you can use the if statement to execute a block of code if a logical condition is true. You can also use the optional else clause to execute a different block of code if the condition is false. The syntax for the if statement in Python is as follows: 

              
                if condition:
                  statement_1
               else:
                  statement_2

The condition can be any expression that evaluates to True or False. If the condition is True, statement_1 is executed; otherwise, statement_2 is executed. Both statement_1 and statement_2 can be any valid Python statements, including nested if statements.

To handle multiple conditions, you can use the elif (short for "else if") keyword. Here's an example:

              
                if condition_1:
                    statement_1
                elif condition_2:
                    statement_2
                elif condition_n:
                    statement_n
                else:
                    statement_last1
                    statement_last2

In this case, only the block of code associated with the first True condition will be executed. You can execute multiple statements within a block,using indentation. It's recommended to use indentation consistently and always include a block statement even if there's only one statement

It's generally advised not to use simple assignments in conditional expressions, as it can be confusing. Instead, use additional parentheses to make the intention clear. For example:

              
                if (x == y):
                    # statements here

While Statement

In Python, the while statement is used to repeatedly execute a block of code as long as a specified condition is true. Here's how you can rephrase the explanation for Python: 

                
                    while condition:
                        statement

The condition is checked before executing the statement inside the loop. If the condition is true, the statement is executed, and then the condition is checked again. This process continues until the condition becomes false. Once the condition is false, control moves to the statement following the while loop.

To execute multiple statements, you can use a block statement (indicated by indentation) to group those statements together.

                
                    n = 0
                    x = 0
                    while n < 3:
                        n += 1
                        x += n

In each iteration, the loop increments the value of n and adds it to x. After three passes, the loop terminates because the condition n < 3 becomes false. The values of n and x will be as follows: After the first pass: n = 1 and x = 1 After the second pass: n = 2 and x = 3 After the third pass: n = 3 and x = 6 Once the loop is complete, the execution continues with the statement following the while loop.

Function Declaration

A function in Python is defined using the def keyword, followed by the function's name and a set of parentheses containing any arguments. The function's statements are written within an indented block. For example, to define a function called square that calculates the square of a number, you can use the following syntax: 

                
                  def square(number):
                    return number * number

In this case, the function takes one argument, number, and the return statement calculates the square of the number and returns the result. When calling the function, you can pass a value to the number parameter, and the function will perform the necessary computations.
